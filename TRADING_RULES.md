## 2. IC（Investment Committee）流程

**频率**: 每小时整点后5分钟
**流程**: Ace同时 @King @Queen @Jack → 三人并行独立分析 → 各自 [DONE] @Ace → Ace综合决策

### 分工
1. **King**: 技术面 — 价格行为 + RSI + MA/BB/MACD + K线形态 + 趋势判断
2. **Queen**: 情绪面 — F&G Index + 散户/机构情绪 + 风险评估
3. **Jack**: 宏观面 — X.com最新crypto新闻 + 地缘/政策/ETF/监管动态

### 铁律
- ⚠️ **必须等三方全部 [DONE] 后再出CEO决策**
- **CEO先有立场，团队来验证** — 不是团队说什么就是什么
- 三方互不参考，各自出独立建议
- IC汇报 < 10行，关键数值+结论+建议操作

## 4. 止盈策略 — 方案C v1.1

> 2026-03-14 Kingo提议讨论，三方+CEO定稿

### 4.1 止盈阶梯（稳健仓LONG）

| 盈利 | 动作 |
|------|------|
| +1.5% | 止损移到成本价（保本） |
| +3% | 平1/3仓（落袋） |
| +4% | 再平1/3仓 |
| 剩余1/3 | 移动止损跟踪 |

### 4.2 移动止损间距（Jack宏观驱动）

| 宏观状态 | 间距 |
|----------|------|
| 强叙事持续（避险+ETF连续净流入） | 2% |
| 中性/无催化 | 1.5% |
| 出现利空 | 1% |

### 4.3 Jack止盈加速条款
- Jack上报重大利空 → 立即平半仓（不等档位触发）
- ETF转净流出 → 间距收窄到1%

### 4.4 Queen风控补充条款
- **最大持仓时间**: 稳健仓48h / 激进仓24h → 触发强制审查（不自动平仓）
  - 盈利>+3%且趋势完好 → 续持，移动止损上移，24h后再审
  - 盈利0~+3%趋势走平 → 平仓
  - 浮亏 → 立即平仓
- **FG极端反转**: 持仓中FG从<20升至>50 → 立即平仓
- **假突破保护**: 触及+2%但回落到成本价，发生2次 → 强制平仓
- **夜间持仓检查**: 睡前查盈亏/FG/利空状态

## 6. 激进仓规则

### 6.1 SOL 激进仓（5x）
**入场条件:**
1. SOL RSI 50-58区间
2. BTC 站稳 MA7
3. Jack 有明确强多催化剂
4. BB 爆破确认
5. 有回踩确认更好（不强制）

**风控**: SL -1.5% / TP +6% / 5x
**核心**: SOL 看爆破

### 6.2 BNB 激进仓（5x）
**入场条件（同时满足）:**
1. BNB RSI(1h): 52~60
2. BTC 站稳 MA7
3. MACD: 金叉或金叉扩张中
4. BB: 中上轨强势
5. Jack: 偏多或不偏空

**风控**: SL -1.5% / TP +6% / 5x
**核心**: BNB 看稳中转强

### 6.3 激进仓通用规则
- 单批 10%（约14-15U），最多2批
- 每日最多开仓1次
- **周末不开激进仓**
- 累计亏损>10U → 暂停
- 账户<130U → 激进仓全停

## 7. 做空策略 V1.2

> 2026-03-13 Kingo批准，三方二审通过

### 7.1 稳健空仓（ETH 2x）
**入场条件（全部满足）:**
1. ETH 跌破 MA7 且 MA7 < MA25（空头排列）
2. MACD Hist 连续2根扩张（负值加大）
3. RSI < 48
4. BTC 跌破 MA7 且 MACD Hist 为负

**风控**: SL +2% / TP -4% / 2x / MARKET平仓

### 7.2 激进空仓（SOL/BNB 2x）
**入场条件（全部满足）:**
1. 跌破 MA7 + BB 下轨突破确认
2. MACD Hist 负值加速
3. RSI < 42
4. Jack 有量化利空催化剂

**风控**: SL +1.5% / TP -6% / 2x / MARKET平仓

### 7.3 做空红线（铁律）
- 🚫 FG > 50 → 禁止做空
- 🚫 FG < 20 → 禁止做空（轧空风险）
- 🚫 BTC 站稳 MA25 以上 → 禁止做空
- 🚫 周末不开空仓
- 🚫 白宫/监管正面表态 24h内不做空
- 🚫 ETF 当日净流入 → 不开空
- 🚫 非农/CPI 发布前后 1h 禁止开空
- 🚫 持有同标的多仓时禁止开空
- 🚫 做空不加仓

### 7.4 做空熔断
- 单日做空亏损 > 5U → 暂停
- 累计亏损 > 8U → 暂停重评
- 连续亏损2次（同日）→ 暂停等第二天
- 账户 < 140U → 禁止做空

### 7.5 轧空防护
- 做空期间 FG 相对入场时上涨 >20点 → 立即平仓
